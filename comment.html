Error after Submitting the Report

Simply go to Yellow Fox again and log in with your credentials. Then click on Report and then Manage Reports. If there is a number in parentheses in the In Progress tab, the processing in Yellow Fox is not completely finished. If there is no number, the latest entry must be downloaded via display (it is important to note that the file is a CSV file). This must then be placed in the csv_file folder which will be in the folder where the executable file is also located. Place the CSV file there and click Next in the user interface.

Error before Submitting the Report

Here it is a bit more complicated. Log in to Yellow Fox again. Then click on Report and then New Report. You can enter anything in the report name. For the report type, you must select Daily Travel Report. For the date range, select the period you want to evaluate. For the vehicles, each vehicle must be selected except for the virtual ones. VERY IMPORTANT: CSV MUST BE SELECTED FOR THE FORMAT. It can now be submitted.
